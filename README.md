## 说 明
    如果你们有什么需要集成的插件，然后自己做比较困难的，可以告诉我，我会在这个例子中添加进去的。
    因为这个项目中集成了很多插件，你们aot build的时候可能会发现chunk很大，但是不要担心，你们实际用
    的时候肯定不会把这些插件都用到的，对不对？
## 注意
    有些插件在对应的分支中才会存在，所以你们一定要看清楚啊。

## 安装&运行
    npm install -g cnpm
    cnpm install -g @angular/cli
    cnpm install
    ng serve
